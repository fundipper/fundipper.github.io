站点设置

2021-01-01

该项为必选置项，map类型，包含站点名称，主题，端口，域名等信息

在项目目录 config.toml，可配置该项

[site]

站点配置

[site]
name = "Fungo"
theme = "fungo"
size = 10
port = ":3000"
origin = "https://fungo.dev"

Notice:

该项所定义内容，在任意模板页均可使用
该项为字典类型，支持配置多个非重名[name]配置项

name

设置站点标题

theme

设置当前使用的主题名称

Notice:

必须为theme目录下存在的主题名称，否则会引发异常

size

设置列表每页项目数据

port

设置预览模式使用端口

origin

设置上线运行使用域名
