新建文章

2021-01-01

file

该指令用于新建文章，在项目目录执行

fungo file article your-article-name

当提示create new file success时，即为执行成功

进入目录

➜  cd your-site-name/content

使用 tree 命令查看目录

➜ tree -L 1
.
└── your-article-name.md

使用 cat 命令查看文件

➜ cat your-article-name.md

---
title: your-article-name
description: ""
category: ""
date: "2022-03-30"
keywords: ""
lang: ""
slug: ""
tag:
    - ""
    - ""
    - ""
---

文章由meta和content两部分构成

默认使用source/article.yaml中内容生成meta信息
用户可自行修改yaml内容，用以实现个性化meta定义

content信息为用户撰写的 markdown

title

文章名称，默认为当前文件名，可自行修改

description

文章描述，可不设置，主要用于SEO

keywords

文章关键词，可不设置，主要用于SEO

category

文章分类

设置后可通过/category/your-category-name/，访问该分类下所有文章

date

文章日期，默认为当前日期

设置后可通过/archive/your-archive-date/，访问该归档下所有文章

lang

文章语言，可不设置，主要用于国际化

lang需对应为国际化配置的语言列表

假设文件路径为post/your-article-name.md

当lang = ""时, 访问地址为:

/post/your-article-name/

当lang = "zh"时, 访问地址为:

/zh/post/your-article-name/

若需同时支持多个预言版本时，因文件名不可重复

建议设置为如下模式:

默认语言

/post/your-article-name.md

中文版本

/post/your-article-name.zh.md

英文版本

/post/your-article-name.en.md

slug

文章访问路径，可不设置，主要用于SEO

假设文件路径为post/your-article-name.md

当slug = ""时, 访问地址为:

/post/your-article-name/

当slug = "this-is-a-slug-url"时, 访问地址为:

/post/this-is-a-slug-url/

当slug为空时，则文章URL为文件名

当slug非空时，则文章URL为slug

相同目录下文件的slug不可重复

tag

文章便签，可设置多个

设置后可通过/tag/your-tag-name/，访问该标签下所有文章

设置文章列表，查看文章配置
