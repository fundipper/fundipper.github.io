在项目目录 config.toml，可配置该项

Notice:

该项为列表类型，支持配置多个非重名[name]配置项

name

设置文档存储目录和渲染模板名称

该目录需自行创建，名称可自定义

该目录支持多级子目录

sidebar

设置文档侧边栏

map[string]interface{}类型，支持多级嵌套

number: 可自定义

name: 可自定义

route: 需与文件真实路径一致

例如:

content/doc/why.md              ->  /doc/why/

content/doc/start/install.md    ->  /doc/start/install/  

content/doc/start/cmd/site.md   ->  /doc/start/cmd/site/

node: 可嵌套多个层级

1 级

sidebar = [
    { number = "1", name = "为什么", route = "/doc/why/" },
    { number = "2", name = "概览", route = "/doc/overview/" },
]

2 级

sidebar = [
    { number = "1", name = "概览", node = [
        { name = "Why Fungo？", route = "/doc/why/" },
        { name = "概览", route = "/doc/overview/" },
    ] },
]

3 级

sidebar = [
    { number = "2", name = "起步", node = [
        { name = "安装", route = "/doc/start/install/" },
        { name = "指令", node = [
            { name = "新建网站", route = "/doc/start/cmd/site/" },
            { name = "新建主题", route = "/doc/start/cmd/theme/" },
            { name = "新建文件", route = "/doc/start/cmd/file/" },
            { name = "预览模式", route = "/doc/start/cmd/serve/" },
            { name = "编译模式", route = "/doc/start/cmd/build/" },
        ] },
    ] },
]
