部署到 Github Pages

2021-01-01

新建SSH

  1. 生成密钥
ssh-keygen -t rsa -m pem -b 4096 -C " your git email name " -N " "

获取私钥

cat .ssh/id_rsa

获取公钥

cat .ssh/id_rsa.pub

  1. 配置私钥
open https://{your-github-page-resource}/settings/secrets/actions/new

Set Name

  ACTIONS_DEPLOY_KEY

Set Value

  input private key
  1. 配置公钥
open https://{your-github-page-path}/settings/keys/new

Set Name

  ACTIONS_DEPLOY_KEY

Set Value

  input public key

Check Allow write access

新建workflows

在项目根目录增加.github/workflows/deploy.yaml文件，然后配置如下信息

使用 docker

因依赖较少，故速度更快

# 名称，可自定义
name: Fungo Deploy

# 依赖
on:
  # 当命令为 push
  push:
    # 且分支为 master 时，触发
    branches:
      - master

  # 当命令为 pull
  pull_request:
    # 且分支为 master 时，触发
    branches:
      - master
  # 手动触发
  workflow_dispatch:

# 任务
jobs:
  # 任务名称，可自定义
  deploy:
    # 启动依赖，可设置为 ubuntu，windows，macos
    runs-on: ubuntu-latest
    
    # 使用docker 镜像
    container:
      image: fundipper/fungo:latest

    # 步骤
    steps:

      # 检出代码
      - name: checkout
        uses: actions/checkout@v3

      # 配置ssh
      - name: config
        run: |
          set -e
          echo "Host *
            StrictHostKeyChecking no
            UserKnownHostsFile=/dev/null
          " > /etc/ssh/ssh_config

      # 编译项目
      - name: build
        run: fungo build

      # 部署项目
      - name: deploy
        uses: peaceiris/actions-gh-pages@v3
        with:
          commit_message: ${{ github.event.head_commit.message }}
          # 部署密钥
          deploy_key: ${{ secrets.ACTIONS_DEPLOY_KEY }}
          # 发布地址
          external_repository: "your github page path as 'fundipper/fundipper.github.io'"
          # 使用分支
          publish_branch: "your branch as 'master'"
          # 使用目录
          publish_dir: ./public
          # 自定义域名
          cname: fungo.dev

使用 go install

因依赖较多，故部署略慢

# 名称，可自定义
name: Fungo Deploy

# 依赖
on:
  # 当命令为 push
  push:
    # 且分支为 master 时，触发
    branches:
      - master

  # 当命令为 pull
  pull_request:
    # 且分支为 master 时，触发
    branches:
      - master
  # 手动触发
  workflow_dispatch:

# 任务
jobs:
  # 任务名称，可自定义
  deploy:
    # 启动依赖，可设置为 ubuntu，windows，macos
    runs-on: ubuntu-latest
    
    # 步骤
    steps:

      # 检出代码
      - name: checkout
        uses: actions/checkout@v3

      # 安装golang
      - name: setup
        uses: actions/setup-go@v3
        with:
          go-version: 1.18

      # 安装fungo
      - name: install
        run: go install github.com/fundipper/fungo@latest

      # 编译项目
      - name: build
        run: fungo build

      # 部署项目
      - name: deploy
        uses: peaceiris/actions-gh-pages@v3
        with:
          commit_message: ${{ github.event.head_commit.message }}
          # 部署密钥
          deploy_key: ${{ secrets.ACTIONS_DEPLOY_KEY }}
          # 发布地址
          external_repository: "your github page path as 'fundipper/fundipper.github.io'"
          # 使用分支
          publish_branch: "your branch as 'master'"
          # 使用目录
          publish_dir: ./public
          # 自定义域名
          cname: fungo.dev

将需要绑定的域名，使用CNAME解析至{your-github-page-path}

  1. 绑定域名

待解析生效后，访问https://{your-github-page-path}/settings/pages
