部署到 Cloudflare Pages

2021-01-01

因 cloudflare 暂未内置 fungo，且使用无法自定义构建环境
故使用 github pages 构建，cloudflare pages 同步部署
请先配置github pages

注册帐号

  1. 打开 https://pages.cloudflare.com/
    注册帐号

部署项目

  1. 创建项目

    打开 https://dash.cloudflare.com/{your-dash-id}/pages/new/provider/github
    创建项目

  2. 关联帐号

    输入 github 帐号，并选择存储库 {your-github-page-path}

  3. 构建部署

    因无法自定义构建环境，故全部使用默认配置，不构建，仅同步

    框架预设: Node（默认）

    构建命令： （默认）

    构建目录： （默认）

  4. 访问项目

    打开 {your-github-page-path}.pages.dev，即可访问项目

绑定域名

需先将域名DNS解析，替换为

  1. 更改DNS

    打开 https://dash.cloudflare.com/{your-dash-id}/add-site
    添加跟域名，然后更改DNS

  2. 配置域名

    打开 https://dash.cloudflare.com/{your-dash-id}/pages/view/{your-site-name}/domains
    设置自定义域名

网站加速

  1. 添加规则

    打开 https://dash.cloudflare.com/{your-dash-id}/{your-site-host}/rules/form
    添加规则

  2. 配置规则

    URL: {your-site-host}/*

    Setting: 缓存级别/缓存所有内容
