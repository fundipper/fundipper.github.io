本机运行

2022-01-01

本机运行，需先安装 golang 环境

安装

go install github.com/fundipper/fungo@latest

命令

fungo 简单易用，仅有 5 个指令

  1. site

    fungo site your-site-name

  2. file

    fungo file your-file-model your-file-name

    • your-file-model 默认可选 [article, document, page]

    • 可通过资源管理，自行编辑 article，document，page，也可通过拓展资源，自行扩展

  3. theme

    fungo theme your-theme-name
    • 若无需开发模板，可以跳过

  4. serve

    fungo serve

  5. build

    fungo build
